📂 Python File Organizer

A simple and efficient Python script that automatically organizes files in a selected directory into separate folders based on their file extensions. This project is built using Python's built-in os and shutil modules, making it lightweight and easy to use.

🚀 Features
Automatically organizes files by extension.
Creates folders if they don't already exist.
Supports multiple file types.
Prevents manual file sorting.
Simple and beginner-friendly code.
Built using only Python's standard library.
🛠️ Technologies Used
Python 3
os module
shutil module
📁 Example

Before:
